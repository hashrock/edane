# 現状の機能

- 基本的なツリーエディタ

# 非機能要件

- [x] コピペの挿入位置をわかりやすくする（ペースト/子追加した新規ノードを破線アンバー枠で1.6秒ハイライト）
- [x] 文字選択と区別したノード選択の方法を用意する（既存のShift+矢印/ドラッグによるノード選択＝全体矩形ハイライトで充足。単一クリックで即編集の即時性は維持）
- [x] アンドゥ、リドゥの挙動を安定させる（Cmd+Shift+Zでredoが発火しないバグを修正＋Cmd+Y対応。node/browserテスト追加）

# 新機能

- [x] R2への画像アップロード機能。ユーザごとに最大10MBの画像アップロードを可能にする。
- [x] 画像管理を可能にする。D1にも画像のメタデータを保存する。
- [x] リンクノードは、titleをfetchし、またfaviconを表示する
- [x] プロジェクト設定画面（/settings：画像管理・使用量・APIトークン）
- [x] 右クリックで、テキストノードの書式変更（フォントサイズ、太字など）

# むずそうな機能

- [ ] カスタムカードを定義できる。スキーマを適用したカラムを持つカードを作ることができる。これにより、例えばWebページのFeaturesカードのような表現をすることができる。
- [ ] 各ノードをJSONとしてWeb公開する機能（CORS対応）
