# むずそうな機能

- [x] カスタムカードを定義できる。スキーマを適用したカラムを持つカードを作ることができる。これにより、例えばWebページのFeaturesカードのような表現をすることができる。
  - スキーマレスの「オブジェクトカード」（`object`ノード）として実装。子ノードの `key: value` テキストがそのままフィールド行としてカード描画される（カード＝サブツリーのビュー。テキストに戻しても情報は失われない）。値は数値/URL/画像/日付/文字列を推論して描画し、数値はカンマ・通貨（¥）・パーセント・小数点桁数の表示形式を右クリックで指定できる。カード上でEnterすると直前カードのキーをプリフィルした兄弟カードを作成（schema by example）。
- [ ] 各ノードをJSONとしてWeb公開する機能（CORS対応）
