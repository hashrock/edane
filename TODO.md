# 現状の機能

- 基本的なツリーエディタ

# 非機能要件

- [ ] コピペの挿入位置をわかりやすくする
- [ ] 文字選択と区別したノード選択の方法を用意する（ただし、文字入力の即時性を維持）
- [ ] アンドゥ、リドゥの挙動を安定させる（E2Eテストを書く）

# 新機能

- [ ] R2への画像アップロード機能。ユーザごとに最大10MBの画像アップロードを可能にする。
- [ ] 画像管理を可能にする。D1にも画像のメタデータを保存する。
- [ ] リンクノードは、titleをfetchし、またfaviconを表示する
- [ ] プロジェクト設定画面
- [ ] 右クリックで、テキストノードの書式変更（フォントサイズ、太字など）

# むずそうな機能

- [ ] カスタムカードを定義できる。スキーマを適用したカラムを持つカードを作ることができる。これにより、例えばWebページのFeaturesカードのような表現をすることができる。
- [ ] 各ノードをJSONとしてWeb公開する機能（CORS対応）
